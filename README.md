# Snake Game (Terminal)

Classic Snake in the terminal using Python's built-in `curses`.

## Run

```bash
uv run python -m snake_game
```

## Run (Pygame UI)

```bash
uv run python -m snake_game.pygame_ui
```

## Tests

```bash
uv run pytest
```

## Lint/Format

```bash
uv run ruff check .
uv run ruff format .
```

## Controls

- Arrows / WASD: move
- P: pause
- R: restart
- Q: quit
